# Full-Stack Travel Booking Platform MVP

A monolithic, extremely fast full-stack travel booking application built in under 30 minutes. 

## Features
- **Frontend**: React, React Router, Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express, MongoDB
- **Integrations**: Razorpay Mock Payments
- **Modules**: Hotel, Flight, Bus, Train, Cab, Bookings, Wishlist, Admin, Auth.

## Setup Instructions

### 1. Backend
```bash
cd backend
npm install
npm start
```
*Note: Ensure MongoDB is running locally on `localhost:27017` or update the `.env.example` as `.env` and set your URI.*

### 2. Frontend
```bash
cd frontend
npm install
npm start
```
*Note: Make sure to copy `.env.example` to `.env` if needed, defaults are provided.*
